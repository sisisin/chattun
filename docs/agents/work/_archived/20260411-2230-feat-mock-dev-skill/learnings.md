# Learnings: feat/mock-dev-skill

## 学び

### SKILL.md の変数展開
- SKILL.md 内のコードブロックでシングルクォート `'$ARGUMENTS'` を使うとClaude Codeが変数を展開しない
- ダブルクォート `"$ARGUMENTS"` を使うか、手順セクションで明示的に変数操作を記述する必要がある

### Skill の手順は具体的に
- Claudeが実行する手順は曖昧にせず、具体的なコマンドを記述する方がよい
- 「ブラウザで確認」のような記述はClaude がツールを持っているかに依存するため、「ユーザーに案内する」のように明確にする
