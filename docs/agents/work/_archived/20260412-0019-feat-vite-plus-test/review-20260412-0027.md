### 指摘事項

1. **[front/vite.config.ts:14]**: `test.exclude` の設定がVitestのデフォルト値（`["**/node_modules/**", "**/.git/**"]`）を上書きしている。カスタムで `exclude` を指定するとデフォルトが消えるため、現在の設定 `['.blueprints/**', 'node_modules/**']` には `**/.git/**` が含まれていない。また `node_modules/**` はグロブの書き方として `**/node_modules/**` の方がネストしたnode_modulesも除外できる。

   修正案: デフォルト除外パターンを引き継ぎつつ追加する形にする。
   ```typescript
   import { configDefaults } from 'vite-plus';

   test: {
     exclude: [...configDefaults.exclude, '.blueprints/**'],
     environment: 'jsdom',
     globals: true,
   },
   ```
   なお `node_modules/**` はVitestのデフォルトに含まれているため個別指定は不要になる。
