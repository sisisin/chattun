# Task: channel matchのバツボタンが小さい + デザインガイド追加

## 背景

設定画面の Channel Match / ミュート設定で、✕（削除）ボタンが 32x32px と小さく、タップ/クリックしづらい。
この修正に合わせて、コンポーネントのサイズや余白に関するデザインガイドラインを策定する。

## 達成条件

1. `.button-remove` のサイズを input/select の高さに揃え、タッチターゲットとして十分なサイズ（36px以上）にする
2. デザインガイド (`docs/guide/ui-design.md`) に「コンポーネントサイズ」セクションを追加し、以下を定義する:
   - インタラクティブ要素の最小サイズ（タッチターゲット）
   - ボタンのサイズバリエーション（primary, remove, add）
   - フォーム行内の要素の高さ揃え方針
3. vp check が通ること
