# Task: モックサーバーに /api/file エンドポイントを追加

## 原文

- Task: モックサーバーに /api/file エンドポイントを追加し、モック環境でも画像プロキシが動作するようにする
  - 現状モックプリセットの画像が /api/file プロキシ経由で404になり、ブラウザ検証できない

## 達成条件

1. モックサーバーに `/api/file` エンドポイントを追加
2. `target_url` クエリパラメータがモックアセットパス（`/api/mock/assets/...`）の場合、該当アセットファイルを返す
3. モックプリセットの画像付きメッセージ（`message-with-image` 等）で画像が正しく表示される

## スコープ

- `server/mock-server/index.ts` にエンドポイント追加のみ
- 本番の `/api/file` と同じインターフェース（`?target_url=...`）を使う
