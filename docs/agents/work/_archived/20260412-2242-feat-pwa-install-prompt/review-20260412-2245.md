### 指摘事項

1. **[front/src/app/components/PwaInstallBanner.tsx:33-34]**: `handleInstall` でインストールが `accepted` になった場合のみ `deferredPrompt` をクリアしているが、`dismissed` の場合は `deferredPrompt` が残ったままバナーが表示され続ける。ユーザーがOSのプロンプトを「キャンセル」した場合でもバナーが再表示されてしまう（同一セッション内で無限に再表示できる）。`result.outcome === 'dismissed'` の場合も `setDeferredPrompt(null)` するか、あるいはセッション内での再表示防止のために `setDismissed(true)` する対応が必要。

2. **[front/src/app/components/PwaInstallBanner.css:5]**: `z-index: 9998` はハードコードされた数値。このプロジェクトでは他の固定 z-index 値との整合性を確認する必要がある。ただしデザインガイドに z-index トークンの定義がないため、既存コードとの一貫性さえ取れていれば許容範囲内。この点は確認のみで必須修正ではない。

3. **[front/src/app/components/PwaInstallBanner.tsx:51]**: 閉じるボタン（`✕`）の `padding: 0 var(--spacing-xs)` はデザインガイドの「インタラクティブ要素の最小タッチターゲットは 36px × 36px」を満たさない可能性が高い。`font-size: var(--font-small)`（0.8rem）かつ padding が 4px のみでは 36px に届かない。タッチデバイスでの操作性が低下するため、最低限 `min-width: 36px; min-height: 36px` を設定するか、`.button-remove` クラスのスタイルを再利用することを検討すること。
