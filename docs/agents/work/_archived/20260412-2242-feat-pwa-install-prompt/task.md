# Task: PWAのアプリとしてインストールされてなかったときにユーザーにインストールを促すようにしたい

## 達成条件

- PWAとしてインストールされていない状態でアクセスしたときに、インストールを促すUIが表示されること
- 邪魔にならない控えめなUIであること
- ユーザーが閉じたら再表示されないこと（セッション内、またはlocalStorageで記憶）
- インストール済みの場合は表示されないこと

## 背景

PWAとしてインストール可能なアプリだが、ユーザーがインストール機能に気づかない可能性がある。邪魔にならない方法でインストールを促したい。

## スコープ

- `beforeinstallprompt` イベントを利用したインストールバナー/ボタンの実装
- 控えめなUI（小さなバナーやトーストなど）
