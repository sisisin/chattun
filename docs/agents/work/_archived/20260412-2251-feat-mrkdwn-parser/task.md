# Task: mrkdwn（slackの記法）をパースしてコンポーネントに落とすレイヤーを実装する

## 達成条件

- mrkdwnテキストをパースしてASTに変換するパーサーが実装されていること
- ASTをReactコンポーネントに展開するレンダラーが実装されていること
- TDDで実装されていること（パーサーのテストが充実していること）
- このタスクで対応するスコープ:
  - プレーンテキスト
  - 太字 (`*bold*`)
  - イタリック (`_italic_`)
  - 打ち消し線 (`~strike~`)
  - インラインコード (`` `code` ``)
  - コードブロック (` ```code``` `)
  - 改行
- 既存の表示パイプラインへの統合は後続タスクで行う

## 背景

現在はtextToHtml()でHTML文字列を生成しdangerouslySetInnerHTMLで表示しているが、
Reactコンポーネントに展開する方式に移行したい。
また、markdown-itで標準Markdownとして処理しているため、
mrkdwn固有の構文（`*bold*`が太字など）が正しく解釈されていない。

## スコープ

- mrkdwnパーサー（テキスト → AST）の実装とテスト
- AST → Reactコンポーネントのレンダラー実装
- 既存パイプラインへの統合は行わない（後続タスクとして追加する）
