### 問題点

1. **空状態のロゴ位置がヘッダーから十分に離れていない（タイムライン・スレッド両方）**
   - 期待: `padding-top: 20vh` = 240px のスペースがヘッダー下に確保される
   - 実際: `padding-top` の computed value が `48px`（= `--spacing-2xl`）になっている
   - 原因: Menu.module.css の `& + div, & + ul, & + a { padding-top: var(--spacing-2xl) }` ルール（CSS nesting）が `._menu_c29jd_1 + div` として展開され、`.tweetlist-empty`（div 要素）の `padding-top: 20vh` を上書きしている
   - DOM構造: `ul._menu` の直後の兄弟要素が `div._tweetlist-empty` であるため、`& + div` セレクタがマッチする
   - 結果: ロゴがヘッダー直下（48pxのみの余白）に表示され、画面のやや上寄り（40%付近）ではなくヘッダーに張り付いた位置になっている
   - スレッド画面（/thread/xxx/yyy）の空状態でも同一の問題が再現
