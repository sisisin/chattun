# Task: スレッドかどうかを視覚的にわかりやすくする

## 達成条件

- タイムライン上でスレッドリプライの投稿が視覚的に区別できる
- 通常のメッセージとの違いが一目でわかるデザイン

## 背景

今はタイムライン画面でスレッドリプライも親メッセージも平たく表示されていて区別がつかない。

## スコープ

- `Tweet.tsx` で `threadTs` の有無によりCSSクラスを出し分け
- `Tweet.css` にスレッドリプライ用のスタイルを追加（左ボーダーで区別）
