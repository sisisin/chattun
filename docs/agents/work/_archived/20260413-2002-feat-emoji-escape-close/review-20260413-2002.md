- 表示崩れの有無: なし

### 問題点

1. **Escキー1回で絵文字ピッカーが閉じない**
   - emoji-mart は Shadow DOM (`em-emoji-picker` カスタム要素) を使用しており、`autoFocus` により検索バーにフォーカスが当たる
   - Shadow DOM 内部で Escape の keydown イベントが `stopPropagation()` されているため、`document.addEventListener('keydown', ...)` のバブルフェーズリスナーに到達しない
   - 検証: キャプチャフェーズでは Escape イベントが `document` に到達することを確認済み（target: `EM-EMOJI-PICKER`）。1回目の Esc で検索バーのフォーカスが外れ、2回目の Esc は target が `BODY` になりバブルフェーズにも到達しピッカーが閉じる
   - `document.dispatchEvent(new KeyboardEvent('keydown', { key: 'Escape' }))` で手動発火した場合は即座に閉じることも確認（コードのロジック自体は正しい）
   - **修正案**: `document.addEventListener('keydown', handleKeyDown, true)` のようにキャプチャフェーズ（第3引数 `true`）でリッスンするよう変更すれば、1回のEscで閉じるようになる。cleanup も `document.removeEventListener('keydown', handleKeyDown, true)` に合わせる必要あり
