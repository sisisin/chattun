# Task: リアクションした人をマウスホバーした瞬間に出したい

## 原文

- Task: リアクションした人をマウスホバーした瞬間に出したい。あとホバーしたときの表示名を投稿の表示名と同じになるようにしたい
  - slack appだとこういうのがホバーしてすぐ出てくる tmp/examples/SCR-20260413-npot.png
  - ネイティブのhoverじゃ難しいならコンポーネントを実装してかまわない

## 分析

### 現状
- リアクションのツールチップにネイティブの `title` 属性を使用 → ブラウザ依存の遅延あり（通常0.5〜1秒）
- リアクター名に `users[user]?.name` を使用 → 投稿の表示名（`display_name || real_name`）と異なる

### 修正内容
1. **即時ツールチップ**: CSS-only のカスタムツールチップ（`visibility` + `opacity` で hover 時に即表示）
2. **表示名統一**: リアクター名の取得を `display_name || real_name` に変更

## 達成条件

- リアクションにホバーした瞬間にツールチップが表示される
- ツールチップの表示名が投稿の表示名と同じになる
