# Task: local-reviewのプロセス内にフロントの動作確認エージェントを含める

## 達成条件

- local-reviewスキル（`.claude/commands/start-local-review.md`）を更新し、フロントエンドの動作に関わるdiffが含まれていた場合にブラウザ動作検証サブエージェントを実行するステップを追加する
- 動作検証の結果がレビューログに記載されること
- レビュワーサブエージェントとブラウザ検証サブエージェントのファイル競合に注意する

## 背景

現在のlocal-reviewはコードレビューのみで、実際にブラウザで動作確認をしていない。フロントの変更がある場合はブラウザを使った動作検証を含めることで品質を向上させたい。
