# Task: CSSにリファクタリングした方が良い場所があるかを検討し、あればリファクタリングTaskを追加する

## 達成条件

- CSSの実装とデザインガイドを読み込み、デザインガイドに即した実装をしやすい構造を検討した結果が作業ディレクトリに記録されている
- リファクタリングが必要な箇所がある場合、TODO.mdにTaskとして追加されている

## 背景

デザインガイド（docs/guide/ui-design.md）が作成されたが、既存のCSSがガイドに沿った構造になっているか検証されていない。
ガイドに即した実装がしやすいCSS構造を作れないかを検討する。

## スコープ

- front/src/app/css/ の共通CSSファイルの構造分析
- 各コンポーネントCSSとデザインガイドの照合
- リファクタリング候補の洗い出しとTask化
