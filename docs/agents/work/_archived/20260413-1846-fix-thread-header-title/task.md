# Task: スレッド画面のヘッダ表示が「すべての投稿」なのは誤り

## 原文

- Task: スレッド画面のヘッダ表示が「すべての投稿」なのは誤り

## 分析

### 現状
- `Menu.tsx` でヘッダの中央テキストが `isSetting` のみで分岐しており、スレッド画面でも「すべての投稿」と表示される
- スレッド画面のパスは `/thread/$channelId/$ts`

### 修正方針
- スレッド画面では「スレッド」と表示する
- `location.pathname` でスレッドかどうかを判定する（`/thread/` で始まるかどうか）

## 達成条件

- スレッド画面のヘッダ中央が「スレッド」と表示される
- タイムライン画面では従来通り「すべての投稿」と表示される
- 設定画面では従来通り「設定」と表示される
