### 指摘事項

1. **[front/src/app/components/menu/Menu.tsx:27-34]**: 設定画面でヘッダタイトル「設定」がクリッカブルなリンク（`<Link to="/">`）になってしまう問題がある。

   変更前は `isSetting` で分岐しており、設定画面では `<span>`（非クリッカブル）を表示していた。変更後は `isTimeline` で分岐するため、設定画面（`isTimeline = false`）では `<Link to="/">` がレンダリングされ、「設定」の文字をクリックすると `/` に遷移してしまう。

   設定画面には既に「戻る」ボタン（`menuGoback`）があり、ヘッダタイトルがリンクになることはUXとして不自然。変更前の設定画面の挙動（非クリッカブルなスパン）を維持するよう修正すべき。

   修正案:
   ```tsx
   <li className={styles.menuTimeline}>
     {isTimeline || isSetting ? (
       <span className={styles.menuTimelineLink}>{headerTitle}</span>
     ) : (
       <Link to="/" className={styles.menuTimelineLink}>
         {headerTitle}
       </Link>
     )}
   </li>
   ```
