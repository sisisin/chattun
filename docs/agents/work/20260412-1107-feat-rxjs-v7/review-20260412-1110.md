## Round 1

STATUS: APPROVED

rxjs 6→7 へのアップグレードは適切に実施されている。`toPromise()` から `firstValueFrom()` への移行、`ajax<Body>` への型引数追加、typeless の pnpm patch による `rxjs/internal-compatibility` 削除対応はいずれも正しく、破壊的変更への対処として過不足がない。`firstValueFrom` が空ストリームで `EmptyError` を投げる懸念も、`HttpClient` の `catchError` が必ず `of(...)` で値を流す構造になっているため問題ない。
