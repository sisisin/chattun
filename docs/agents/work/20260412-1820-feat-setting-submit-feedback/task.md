# Task: 設定画面で変更をsubmitしたときにフィードバックが欲しい

## 達成条件
- 設定を変更したとき（select変更、フォームsubmit、チェックボックス切り替え）に保存完了のフィードバックが表示される
- フィードバックは自動的に消える
- 既存の設定保存ロジックに破壊的変更がない

## 背景
- 現在は設定を変更しても視覚的なフィードバックがなく、保存されたかどうかが分かりにくい
- localStorageへの保存は同期的なのでローディングは不要、保存完了の通知のみで十分

## スコープ
- トースト通知コンポーネントを作成し、設定保存時に表示
- `updateSettingFulfilled` アクションをトリガーにトーストを表示
- CSSアニメーションで2秒程度表示後に自動的に消える
