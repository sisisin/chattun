# Task: timelineでスレッド投稿の右下にスレッドページへのリンクを表示させる

## 達成条件

- タイムラインに表示されるスレッド付きメッセージの右下に、スレッドページへ遷移するリンクが表示されること
- リンクをクリックするとスレッドページ（`/thread/:channelId/:ts`）に遷移すること
- スレッドを持たないメッセージにはリンクが表示されないこと

## 背景

タイムラインビューでスレッドがあるメッセージを見たとき、そのスレッドの中身を確認するためにスレッドページへ素早くアクセスしたい。

## スコープ

- タイムラインのメッセージコンポーネントにスレッドリンクを追加
- 既存のスレッドページのルーティングを活用
