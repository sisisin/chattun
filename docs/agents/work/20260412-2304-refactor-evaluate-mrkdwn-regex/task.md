# Task: regexでのmrkdwnのパース処理のメンテナンス性を評価し、必要性が認められればregexをやめるリファクタリングを行う

## 達成条件

- 現在のregexベースパーサーのメンテナンス性を評価した結果が記録されていること
- 必要性が認められた場合: 文字走査ベースのパーサーにリファクタリングされていること
- 必要性がない場合: 現状維持の判断根拠が記録されていること
- 既存テストがすべて通ること

## 背景

前タスクで実装したmrkdwnパーサーはregexベース。今後メンション・絵文字・リンク等の構文を追加していく予定。
regexが複雑化して保守困難にならないか、今の段階で評価しておく。

## スコープ

- parser.tsの現在の実装の評価
- 今後追加予定の構文（メンション、絵文字、リンク）を考慮した拡張性の評価
- 必要であればリファクタリング
