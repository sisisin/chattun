# Task: スレッド画面で「スレッドを表示」をなくして

## 原文

- Task: スレッド画面で「スレッドを表示」をなくして

## 解釈

スレッド画面（`/thread/:channelId/:ts`）では既にスレッド内にいるため、「スレッドを表示」リンクは不要。ThreadView で表示される Tweet に対してこのリンクを非表示にする。

## 達成条件

- スレッド画面で「スレッドを表示」テキストリンクが表示されない
- タイムライン画面では引き続き「スレッドを表示」リンクが表示される

## スコープ

- `Tweet` / `TweetView` に `inThread` prop を追加し、スレッド画面では「スレッドを表示」リンクを非表示にする
- ThreadView から `inThread={true}` を渡す
